## 1. 产品概述

多租户CRM与销售漏斗管理Web系统，帮助企业销售团队高效管理客户关系、跟踪销售线索、推进商机转化，实现从线索到成交的全流程数字化管理。系统支持多租户数据隔离，基于角色的权限控制，适用于中小型销售团队。

- **核心价值**：提升销售团队协作效率，可视化销售漏斗，加速商机转化
- **目标用户**：销售团队管理者、销售人员、企业运营人员
- **部署方式**：浏览器端SPA应用，IndexedDB本地存储，支持多租户切换

## 2. 核心功能

### 2.1 用户角色

| 角色 | 登录方式 | 核心权限 |
|------|----------|----------|
| 超级管理员 | 账号登录 | 管理所有租户、平台配置 |
| 租户管理员 | 账号登录 | 管理本租户成员、配置销售阶段、查看全部数据 |
| 销售经理 | 账号登录 | 查看团队数据、分配线索、管理下属、查看漏斗统计 |
| 销售人员 | 账号登录 | 管理自己的客户与商机、跟进线索、更新商机阶段 |

### 2.2 功能模块

1. **仪表盘**：数据概览卡片、销售漏斗概览、近期跟进提醒、业绩排行、趋势图
2. **客户管理**：客户列表、客户详情、联系人管理、跟进记录、客户等级
3. **线索池**：线索列表、线索分配、线索转化、线索废弃、线索来源
4. **商机管理**：看板视图、表格视图、阶段拖拽、商机详情、阶段变更历史
5. **销售漏斗**：漏斗可视化、数据筛选、健康度分析、漏斗对比
6. **团队成员**：成员列表、角色管理、邀请成员、销售目标设定
7. **系统设置**：销售阶段配置、线索来源配置、客户等级配置、租户基本信息

## 3. 核心流程

### 3.1 销售线索转化流程

线索创建 → 线索分配 → 销售人员跟进 → 确认需求 → 转化为商机 → 阶段推进 → 赢单/输单

```mermaid
flowchart LR
    A["线索创建"] --> B["线索分配"]
    B --> C["销售人员跟进"]
    C --> D["确认需求"]
    D --> E["转化为商机"]
    E --> F["初步接洽"]
    F --> G["需求确认"]
    G --> H["方案/报价"]
    H --> I["商务谈判"]
    I --> J["赢单/输单"]
```

### 3.2 多租户数据隔离流程

用户登录 → 选择租户 → 加载租户数据 → 所有操作基于tenantId隔离 → 登出清空会话

## 4. 用户界面设计

### 4.1 设计风格

**设计理念**：专业商务风格，数据可视化驱动，强调信息层级与操作效率

- **主色调**：深邃蓝 (#1e3a5f) - 体现专业与信任感
- **辅助色**：翡翠绿 (#10b981) - 代表成交与增长；琥珀橙 (#f59e0b) - 代表提醒与跟进
- **强调色**：靛蓝紫 (#6366f1) - 用于关键操作与数据高亮
- **字体**：Inter 作为主要字体，搭配 Space Grotesk 用于数据展示
- **卡片风格**：圆角 12px，微妙阴影，悬停时轻微上浮
- **图标风格**：线性图标，使用 Lucide 图标库
- **布局**：左侧固定侧边栏 + 顶部导航栏 + 内容区域，经典后台布局

### 4.3 响应式设计

- **桌面端**（≥1280px）：完整侧边栏 + 内容区，多列布局
- **平板端**（768px-1279px）：可收起侧边栏，自适应列数
- **移动端**（<768px）：底部导航栏，单列布局，简化操作

### 4.4 动效与交互

- 页面切换：淡入淡出 + 轻微位移
- 卡片悬停：Y轴上浮 4px + 阴影加深
- 数据加载：骨架屏 + 脉冲动画
- 拖拽操作：半透明拖拽预览 + 放置区域高亮
- 数字变化：滚动数字动画
- 按钮交互：按下时轻微缩放 + 涟漪效果
